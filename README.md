# گوسان — Gosan

Website of **Gosan** (گوسان), a Persian-language cultural and arts magazine published by the Gosan Institute for Culture and Art.

Live site: https://yootazi.github.io/gosan-website/

## Structure

- `index.html` — homepage entry point
- `*.jsx`, `*.js` — page components and site behavior (loaded in-browser)
- `site.css`, `site-x.css` — site styles
- `_ds/` — design system (tokens, base styles, components)
- `assets/`, `uploads/` — images and media
- `institute.html` — Gosan Institute page

Static site — no build step. Serve the repo root with any static file server.
